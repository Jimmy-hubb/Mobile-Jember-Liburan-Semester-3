 # 📱 Si Jeli – Jember Liburan

Aplikasi **Si Jeli (Sistem Informasi Jelajah Liburan Jember)** merupakan aplikasi **mobile berbasis Android** yang dikembangkan menggunakan **Java** dan **Android Studio**.

Aplikasi ini dirancang untuk membantu wisatawan menemukan berbagai **destinasi wisata di Kabupaten Jember**, mulai dari wisata alam, wisata budaya, hingga tempat rekreasi populer.

Melalui aplikasi ini, pengguna dapat memperoleh **informasi lokasi wisata, deskripsi tempat wisata, serta rekomendasi tempat liburan di Jember** dengan mudah melalui smartphone.

Project ini dibuat sebagai bagian dari **portfolio pengembangan aplikasi mobile Android** serta pembelajaran dalam membangun aplikasi berbasis mobile menggunakan Android Studio.

---

# 🚀 Features

Beberapa fitur utama dalam aplikasi **Si Jeli Jember Liburan** antara lain:

### 🗺️ Daftar Destinasi Wisata
Menampilkan berbagai daftar tempat wisata yang ada di Kabupaten Jember.

### 📍 Detail Tempat Wisata
Pengguna dapat melihat detail informasi wisata seperti:

- Nama tempat wisata  
- Deskripsi lokasi  
- Gambar wisata  
- Informasi tambahan mengenai destinasi  

### 🔎 Pencarian Tempat Wisata
Memudahkan pengguna untuk mencari tempat wisata yang diinginkan.

### ⭐ Rekomendasi Wisata
Menampilkan beberapa destinasi wisata populer di Jember.

### 📱 User Friendly Interface
Tampilan aplikasi dirancang sederhana dan mudah digunakan oleh pengguna.

# ⚙️ Installation

Ikuti langkah berikut untuk menjalankan project secara lokal.

### 1. Clone Repository

```bash
git clone https://github.com/username/si-jeli-jember-liburan.git
```

### 2. Masuk ke Folder Project

```bash
cd si-jeli-jember-liburan
```

### 3. Buka Project di Android Studio

1. Buka **Android Studio**
2. Klik **Open Project**
3. Pilih folder project **si-jeli-jember-liburan**

### 4. Sync Gradle

Setelah project terbuka lakukan sinkronisasi Gradle:

```
File → Sync Project with Gradle Files
```

Android Studio akan menginstall semua dependency yang dibutuhkan.

### 5. Jalankan Aplikasi

1. Hubungkan perangkat Android atau gunakan emulator  
2. Klik tombol **Run ▶** pada Android Studio
